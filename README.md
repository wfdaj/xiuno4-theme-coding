# xiuno4-theme-coding

适用于 xiuno4 的 coding 样式模板
- - -
## 安装方法
下载得到 xiuno4-theme-coding-master.zip 压缩包，把 xy_coding 文件夹上传到 xiuno 程序 plugin/目录下，后台安装即可。
